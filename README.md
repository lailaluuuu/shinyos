# ShinyOS

> Learn like it’s a game — an interactive, gamified learning platform for foundational subjects.

## Overview
ShinyOS is a browser-based educational prototype that makes learning core subjects engaging and fun. It features a playful UI, XP and streak tracking, and interactive lessons with quizzes. Subjects include economics, investing basics, mind and psychology, space history, and more.

## Features
- **Gamified learning:** Earn XP, keep streaks, and progress through eras.
- **Interactive lessons:** Each subject contains content and quizzes.
- **Modern UI:** Clean, responsive design with a focus on usability.
- **Easy to extend:** Add new subjects and lessons by editing simple JavaScript files.

## Project Structure
- `index.html` — Main HTML entry point
- `style.css` — App styling
- `app.js` — Main app logic, subject/lesson handling
- `data.js` — Investing basics track and lesson data

## Getting Started
1. **Clone or download this repository.**
2. **Open `index.html` in your browser.**
3. Explore the world, select a subject, and start learning!

## Customization
- To add or edit lessons, modify `app.js` (for subjects) and `data.js` (for investing basics track).
- Update styles in `style.css` for a different look and feel.

## Technologies Used
- HTML5, CSS3, JavaScript (no frameworks)
